# Rag Insert Command

**Type:** method  
**Community:** [[community-61]]  
**Degree:** 3

## Description

- A `rag` subcommand for adding new data into the knowledge graph, specifying its source.
- A subcommand of `Rag CLI` for ad-hoc text insertion with an optional source label.

## Related

- [[entities/rag-cli|Rag CLI]] — The `Rag CLI` includes the `Rag Insert Command`.
- [[entities/zod|Zod]] — The `Rag Insert Command` is used to insert a decision related to using `Zod`.
- [[entities/chat-yyyy-mm-dd-source|Chat-YYYY-MM-DD Source]] — The `Rag Insert Command` uses `Chat-YYYY-MM-DD Source` to identify the origin of inserted data.
