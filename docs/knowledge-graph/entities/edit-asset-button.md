# Edit Asset Button

**Type:** artifact  
**Community:** [[community-17]]  
**Degree:** 2

## Description

- A button to initiate the editing process for a selected asset from the assets table.

## Related

- [[entities/selected-asset-id|Selected Asset ID]] — The Edit Asset Button is disabled if no asset is selected (Selected Asset ID is null).
- [[entities/set-edit-asset-method|Set Edit Asset Method]] — Clicking the Edit Asset Button invokes the Set Edit Asset Method to prepare an asset for editing.
