# Next.js Framework

**Type:** concept  
**Community:** [[community-2]]  
**Degree:** 3

## Description

- The web development framework that provides the environment and structure for the `Fixed-Income API Route`.
- A React framework for building web applications, for which `next.config.ts` provides configuration.
