# Rag Local Command

**Type:** method  
**Community:** [[community-61]]

## Description

- A `rag` subcommand used to query the local neighborhood of specific entities within the knowledge graph.
- A subcommand of `Rag CLI` for local mode, focusing on entity neighborhoods.
