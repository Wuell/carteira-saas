# Global Alias

**Type:** concept  
**Community:** [[community-51]]  
**Degree:** 5

## Description

- A type of alias that might cause issues by pointing to an incorrect project.
- A persistent alias defined in a shell configuration file, pointing to a specific executable regardless of the current directory.

## Related

- [[entities/user|User]] — A global alias might point to the wrong project for the user.
- [[entities/repository-switch|Repository Switch]] — A global alias can cause issues when the user switches repositories.
