# Use Client Directive

**Type:** concept  
**Community:** [[community-70]]  
**Degree:** 2

## Description

- The 'use client' directive is a marker in React components, indicating that the component should be rendered on the client side.
- The 'use client' directive is a marker at the top of a file, indicating that the module and its children should be rendered on the client side.
