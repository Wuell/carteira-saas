# Rag Stats Command

**Type:** method  
**Community:** [[community-61]]  
**Degree:** 5

## Description

- A LightRAG command (`rag stats`) used to display statistics about the LightRAG system's current state.
- A `rag` subcommand that displays statistical counts related to the knowledge graph.
- A subcommand of `Rag CLI` to output JSON with entities/relations/docs.
