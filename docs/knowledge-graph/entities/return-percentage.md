# Return Percentage

**Type:** data  
**Community:** [[community-68]]  
**Degree:** 8

## Description

- The percentage gain or loss on an investment.
- A property of an `Investment Lot`, representing the percentage return on the investment.
- A data point showing the percentage return generated by the portfolio's investments.
