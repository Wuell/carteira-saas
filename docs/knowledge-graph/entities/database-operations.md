# Database Operations

**Type:** concept  
**Community:** [[community-2]]  
**Degree:** 6

## Description

- The set of interactions with the database, including finding, creating, updating, and deleting records.

## Related

- [[entities/http-get-method|HTTP GET Method]] — The `HTTP GET Method` performs `Database Operations` to retrieve `FixedIncomeLot Data Model` records.
- [[entities/http-post-method|HTTP POST Method]] — The `HTTP POST Method` performs `Database Operations` to create new `FixedIncomeLot Data Model` records.
- [[entities/http-patch-method|HTTP PATCH Method]] — The `HTTP PATCH Method` performs `Database Operations` to update `FixedIncomeLot Data Model` records.
- [[entities/http-delete-method|HTTP DELETE Method]] — The `HTTP DELETE Method` performs `Database Operations` to delete `FixedIncomeLot Data Model` records.
- [[entities/fixedincomelot-data-model|FixedIncomeLot Data Model]] — The `Database Operations` interact with the `FixedIncomeLot Data Model` for persistence.
- [[entities/user-data-model|User Data Model]] — The `Database Operations` interact with the `User Data Model` to associate fixed income lots.
