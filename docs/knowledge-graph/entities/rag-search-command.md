# Rag Search Command

**Type:** method  
**Community:** [[community-61]]  
**Degree:** 7

## Description

- A LightRAG command (`rag search "explique o projeto em 2 frases"`) used to query the `Knowledge Graph` and retrieve information.
- A `rag` subcommand that performs a hybrid search, providing synthesized answers with citations.
- A subcommand of `Rag CLI` for hybrid queries with synthesis and citations, also the default experience.
