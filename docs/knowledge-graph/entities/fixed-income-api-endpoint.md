# Fixed-Income API Endpoint

**Type:** concept  
**Community:** [[community-76]]  
**Degree:** 4

## Description

- The API endpoint '/api/fixed-income' used for managing fixed-income investment data through PATCH and DELETE requests.

## Related

- [[entities/http-patch-method|HTTP PATCH Method]] — The HTTP PATCH Method is used for requests to the Fixed-Income API Endpoint.
- [[entities/http-delete-method|HTTP DELETE Method]] — The HTTP DELETE Method is used for requests to the Fixed-Income API Endpoint.
- [[entities/edit-fixed-mutation|Edit Fixed Mutation]] — The Edit Fixed Mutation sends a PATCH request to the Fixed-Income API Endpoint to update fixed-income data.
- [[entities/delete-fixed-mutation|Delete Fixed Mutation]] — The Delete Fixed Mutation sends a DELETE request to the Fixed-Income API Endpoint to remove fixed-income data.
