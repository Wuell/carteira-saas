# Noob-Friendly Default

**Type:** concept  
**Community:** [[community-51]]  
**Degree:** 4

## Description

- The principle of designing instructions assuming the user is new to Python, `uv`, and MCP, including auto-installation of prerequisites.
- A guiding principle to assume the user has no prior experience with Python, uv, or MCP, and to handle prerequisites automatically.
