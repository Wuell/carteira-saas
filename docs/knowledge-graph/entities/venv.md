# Venv

**Type:** concept  
**Community:** [[community-60]]  
**Degree:** 4

## Description

- A Python virtual environment, managed by `uv`.
- A Python virtual environment, used to isolate project dependencies and provide access to the `rag` command.

## Related

- [[entities/uv|Uv]] — Uv manages Python virtual environments (venv).
- [[entities/rag-cli|Rag CLI]] — The `Rag CLI` becomes available in the terminal after `Venv` activation.
- [[entities/deactivate-command|Deactivate Command]] — The `Deactivate Command` is used to exit or deactivate the `Venv`.
- [[entities/lightrag-dependencies|LightRAG Dependencies]] — LightRAG dependencies are automatically installed within an isolated virtual environment (`.venv`).
